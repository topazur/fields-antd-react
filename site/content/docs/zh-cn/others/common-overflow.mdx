> 子元素超出后折叠或者隐藏 (在应用层处理好的 dom 结构，通过 content 和 restContent 传入组件，组件内部只管宽度计算逻辑，而不考虑各种未知渲染情景)。

## 使用

```tsx
import { Overflow } from '@odinlin/fields-antd-react/__builtins__/widgets';
```

### 备注

- ❌方案一: [纯 CSS 实现多标签自动显示超出数量](https://juejin.cn/post/7352785768094203913)
- ❌方案二：[Intersection Observer API](https://developer.mozilla.org/zh-CN/docs/Web/API/IntersectionObserver)
- ✅方案三：累加子元素宽度与容器宽度比较大小，来判断是否超出
