> 比如 `@某人`、`#issue` 等场景。如若需选择的详细选项，可从 props.onSelect 获取，包装层未做处理。

## 使用

```tsx
import { Mentions } from '@odinlin/fields-antd-react/__builtins__/widgets';

/** 本地实现拼音搜索示例，远程需要服务端做支持 */
// import { match } from "pinyin-pro"
// 伪代码: props.filterOption = (inputValue, option) => !!match(option.label as string, inputValue)
```

### 备注

> 代码实现基于 [antd.Mentions](https://ant.design/components/mentions-cn)、[rc-mentions](https://github.com/react-component/mentions)
> - feat: 包含组件库所有功能，仅在其基础上扩展远程搜索功能，都是通过新增 props 实现，未覆盖原有属性。
