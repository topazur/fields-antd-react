> 弹出一个下拉菜单给用户选择操作，用于代替原生的选择器，或者需要一个更优雅的多选器时。

## 使用

```tsx
import { Select } from '@odinlin/fields-antd-react/__builtins__/widgets';

/** 本地实现拼音搜索示例，远程需要服务端做支持 */
// import { match } from "pinyin-pro"
// 伪代码: props.filterOption = (inputValue, option) => !!match(option.label as string, inputValue)
```

### 备注

> 代码实现基于 [antd.Select](https://ant.design/components/select-cn)、[rc-select](https://github.com/react-component/select)
> - fix: 解决 rc-select 中 measureRef 是用于测量输入文字长度 scrollWidth，过长时横向出现滚动条问题。
> - feat: 通过 css 覆盖实现 readOnly 效果。
> - feat: 包含组件库所有功能，仅在其基础上扩展远程搜索功能，都是通过新增 props 实现，未覆盖原有属性。
> - feat: 实现 loupeRender 放大镜模式。
> - perf: 对其他组件库是配置时，若其未实现多选响应式，可通过 Overflow 实现。
