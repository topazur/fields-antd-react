> 支持拖拽的双栏穿梭选择框 (基于 AgGrid 跨表格拖拽实现)。

## 使用

```tsx
import { GridTransfer } from '@odinlin/fields-antd-react/__builtins__/widgets';
```

### 备注

> 代码实现基于 [AgGrid.RowDragging 行拖动](https://www.ag-grid.com/react-data-grid/row-dragging)
> - 左边一栏为 source，右边一栏为 target。但是 API 的设计是以 left 和 right 前缀区分，更加通俗易懂。
