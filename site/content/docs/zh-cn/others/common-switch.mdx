> 使用开关切换两种状态之间。

## 使用

```tsx
import { Switch } from '@odinlin/fields-antd-react/__builtins__/widgets';
```

### 备注

> 代码实现基于 [antd.Switch](https://ant.design/components/switch-cn)、[antd.Checkbox](https://ant.design/components/checkbox-cn)
> - 新增 type 属性: 使其在 Switch 和 Checkbox 组件之前切换 (当 type === 'checkbox' 渲染复选框)
> - 新增 activeValue 属性: 支持数据类型(boolean | number | string)，通过来全等来判断
> - 新增 inactiveValue 属性: 支持数据类型(boolean | number | string)，通过来全等来判断
> - 新增 checkedChildren 属性: 选中时的 label 内容
> - 新增 unCheckedChildren 属性: 非选中时的 label 内容
