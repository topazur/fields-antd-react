> 24 栅格系统

## 使用

```tsx
import { Row, Col } from '@odinlin/fields-antd-react/__builtins__/widgets';
```

### 备注

> 代码实现参考于 [antd.Grid](https://ant.design/components/grid-cn)
> - 不同的是响应式设计，该组件是先计算出当前断点下的具体值，再去改变类名或者样式。
